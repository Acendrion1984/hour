GitHub-Sync
Auto-Sync mit privatem GitHub-Repo
So legst du das Repo an:
1. Erstelle auf github.com/new ein privates Repository (z.B. jahresplan-data). Setze einen Haken bei "Add a README" damit der Branch existiert.
2. Erstelle einen Personal Access Token unter Settings → Tokens (fine-grained) mit Zugriff nur auf dieses Repo und Permission Contents: Read and write.
3. Token hier einfügen, Repo + Owner eintragen, Speichern. Die App lädt dann beim Start automatisch und sichert nach jeder Änderung.

Sicherheitshinweis: Der Token wird unverschlüsselt im Browser-Speicher (localStorage) abgelegt. Verwende die App nur auf vertrauenswürdigen Geräten. Bei Verlust den Token in den GitHub-Settings sofort widerrufen.
